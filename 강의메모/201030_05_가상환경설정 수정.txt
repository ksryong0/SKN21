1. 가상환경 설정 및 가상환경 이름을 .venv로 지정
터미널에서 가상환경을 설치할 폴더 경로로 이동한 후,
uv venv .venv --python=3.13

2. 가상환경 활성화
.venv\Scripts\activate

3. VS Code에 가상환경을 python interpretor로 설정(한번만 해주면 됨)
명령팔레트 실행(Ctrl+Shift+P)->Python: Select Interpreter->작업 영역 안에 있는거 선택
주피터 노트북에서 "커널 선택"에서 가상환경을 설정

4. 3rd party 패키지(라이브러리) 설치
uv pip install 패키지명
반드시 uv로 설치해야 uv로 설치한 가상환경에 라이브러리가 install 된다.

ipykernel: 주피터 노트북 실행을 위해 필요
uv pip install ipykernel

uv pip install pandas matplotlib
