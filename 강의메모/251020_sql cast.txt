NULL인 데이터를 WHERE절에서 쓸 때는 '= NULL' 이 아니고 'IS NULL'로 써야함!
NULL이 아닌 데이터를 WHERE절에서 조회할 때는 'IS NOT NULL' 로 써야함.

같지 않다는 != 또는 <> 로 사용가능.

UPPER(컬럼명) 을 하면 결과값을 대문자로 바꿔줌
피연산자가 null인 경우 결과는 null. ex)1+2+3+NULL 하면 결과는 NULL 이다.
같은 컬럼을 여러번 조회할 수 있다.

AND랑 OR가 같이 있으면 연산자 우선순위가 AND가 우선임. 무조건 AND 먼저함.

컬럼의 이름 대신에 컬럼의 순번(SELECT에서 조회할 컬럼의 순번)으로 ORDER BY 가능

밑에 두개는 같은 코드임
SELECT emp_id, emp_name, salary
	FROM emp
    WHERE salary >= 5000
    ORDER BY salary DESC;

SELECT emp_id, emp_name, salary
	FROM emp
    WHERE salary >= 5000
    ORDER BY 3 DESC;
