ORM이란?
member class - member table 매핑해둠.

name이 홍길동인 것을 select -> select 쿼리 결과로 나온 member table의 name을 member class의 name에 저장. 테이블의 age를 클래스의 age에 저장.

ORM의 장점
비지니스 로직과 DB 로직을 분리할 수 있다.
단점
DBMS 고유 기능을 사용할 수 없다.
복잡한 쿼리 처리 어렵.
